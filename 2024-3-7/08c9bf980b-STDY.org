#+TITLE: git format-patch部分选项功能说明          

git format-patch -3 --start-number=0 --thread --subject-prefix="PATCH v3" --cover-letter -o patchtmp
1. 没法挑选几个commit生成一个patch，只能一个commit对应一个patch。
2. 通过cover-letter选项生成的overview需要手动添加-to -cc在文件中。
3. --start-number=0指明patch从0开始索引。
4. --thread保证生成的多个patches：
   over-letter.patch  <-- First patch(committed early)
                                    <-- Second patch
				    <-- left patches
