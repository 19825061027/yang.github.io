* 介绍
1. Asynchronous process execution context
2. An independent thread serves as the asynchronous execution context.
   The queue is called workqueue and the thread is called worker.
3. workqueue上没有任务时，worker进入idle状态。
4. 新任务进入队列，worker重新开始执行。
